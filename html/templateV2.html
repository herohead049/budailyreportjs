Backup Report Site

1. Backup Report Card

The report card report is included in the reports on this site. It shows the last 7 days of backups for each server. It has different colors for each backup type including failures.

2. Summary Failure Report

The Summary Failure Report is included in the reports on this site. This will show a summary of the failed backups for a 24 hour period.
Note:
  • If the report shows "No results" this means that there were no failures during the reporting time.
  • If a failure occurs but was sucessful on the retry, the failure will not be included in this report.
  • Columns with * are sourced from Trico Website. If data is missing or incorrect you should request a change by clicking on Request Trico Change. .
